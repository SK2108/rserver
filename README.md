# Rserver

Building a simple server with Ruby

## Getting started

### Initialise Bundler

```markdown
➜  gem install bundler
➜  bundle init
```

This will create a Gemfile.

Install it by running this command:

```markdown
bundle install
```

### Install Rack

Use the command below:

```markdown
➜  bundle add rack
```

### Launch the server

- Open the ``rack`` folder in your terminal.

- Type this command to start the server:

```markdown
rack
```
